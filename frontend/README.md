# ThreatLens Frontend

React-based dashboard for the ThreatLens threat intelligence platform.

## Setup

```bash
npm install
npm start
```

## Environment Variables
REACT_APP_API_URL=https://threatlens-api-golw.onrender.com

## Pages

- **Dashboard** — Live threat stats, charts, alerts
- **Threats** — Full threat indicator table with search and filter
- **Threat Hunting** — Hunt by tag, MITRE technique, C2 infrastructure
- **ML Detection** — Isolation Forest anomaly detection
- **AI Analyst** — AI-powered threat analysis
